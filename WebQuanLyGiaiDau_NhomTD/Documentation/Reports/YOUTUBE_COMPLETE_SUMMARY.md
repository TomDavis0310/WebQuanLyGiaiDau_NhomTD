## 🔧 **SETUP REQUIREMENTS**

### **🔑 YouTube API Key:**
1. Tạo project trên Google Cloud Console
2. Enable YouTube Data API v3
3. Tạo API key
4. Cập nhật `appsettings.json`:
```json
{
  "YouTube": {
    "ApiKey": "YOUR_YOUTUBE_API_KEY_HERE"
  }
}
```

### **🚀 Deployment:**
- Ứng dụng đã sẵn sàng chạy
